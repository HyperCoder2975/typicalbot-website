Disable Nickname Changing

Settings - Nicknames

A way to disable the use of the nickname command.

  • Disable the nickname command
  • $settings edit nonickname enable
  • Enable the $nickname command
  • $settings edit nonickname disable

Music Default Permissions

Settings - Music

Change the required permission level for users to use all music commands.

  • Allow everyone to use all music commands
  • $settings edit music-permissions all
  • Allow djs to use all music commands
  • $settings edit music-permissions dj
  • Allow moderators to use all music commands
  • $settings edit music-permissions moderator
  • Allow administrators to use all music commands
  • $settings edit music-permissions administrator

Music Play Permissions

Settings - Music

Change the required permission level for users to use the play command. Overrides the music-permission setting when not set to off.

  • Default to the music default permission
  • $settings edit music-play off
  • Allow everyone to use the play command
  • $settings edit music-play all
  • Allow djs to use the play command
  • $settings edit music-play dj
  • Allow moderators to use the play command
  • $settings edit music-play moderator
  • Allow administrators to use the play command
  • $settings edit music-play administrator

Music Time Limit

Settings - Music

Change the maximum time limit for a video (in seconds). Minimum is 120 seconds, maximum is 7200 seconds.

  • Set the time limit to the default
  • $settings edit music-timelimit default
  • Set the timelimit to 320 seconds
  • $settings edit music-timelimit 320

Music Queue Limit

Settings -

Change the maximum queue limit. Maximum is 100 videos.

  • Set the queue limit to the default (10 songs)
  • $settings edit music-queuelimit default
  • Set the queue limit to 5 songs
  • $settings edit music-queuelimit 5
