Administrator roles

Settings - levels

Administrator role that will grant users with the role(s) permission level 3.

  • Make just one role administrator
  • $settings edit adminrole ROLE-NAME-HERE
  • Add role to list of roles with administrator permissions
  • $settings edit adminrole add ROLE-NAME-HERE
  • Remove role from list of roles with administrator permissions
  • $settings edit adminrole remove ROLE-NAME-HERE
  • Remove administrator permissions from all roles in the list
  • $settings edit adminrole clear
  • Disable administrator permissions
  • $settings edit adminrole disable
